Tek465B System Monitor

Tek465B System Monitor is a Small Windows application made in C Sharp to send text including system hardware information to a 16x02 or 20x02 LCD controlled by Arduino/Matrix microncontroller.

Analog_To_PWM

Arduino code that Sample an analog input at 36.5khz and set the PWM on one output synchronically.
There is also a vRat(Voltage Ratio) setting, this code is Untested as of now
